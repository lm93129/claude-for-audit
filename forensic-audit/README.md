# 司法会计/舞弊审计插件（forensic-audit）

覆盖舞弊审计和司法会计——红旗识别、电子数据取证、异常交易分析、本福特定律检验、举报线索分类。

## 技能速查

| 命令 | 说明 |
|------|------|
| `/forensic-audit:cold-start-interview` | 冷启动采访 |
| `/forensic-audit:red-flags` | 舞弊红旗识别 |
| `/forensic-audit:data-forensics` | 电子数据取证 |
| `/forensic-audit:transaction-analysis` | 异常交易分析 |
| `/forensic-audit:benford-analysis` | 本福特定律检验 |
| `/forensic-audit:whistleblower-triage` | 举报线索三色分类 |
