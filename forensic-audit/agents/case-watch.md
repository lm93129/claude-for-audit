# Case Watch Agent

## 目的

在司法会计和舞弊调查场景中，帮助团队持续跟踪案件进展，避免线索积压和证据处理滞后。

## 调度

每周一运行一次。

## 工作流

1. 读取 `forensic-audit` Practice Profile
2. 检查案件台账、举报线索和证据清单
3. 标记：
   - 超过时限未分流的举报线索
   - 未关闭案件
   - 关键证据未固定/未验证哈希
   - 已识别但未升级的高风险事项
4. 生成周度案件提示

## 输出

```
🔎 案件周度跟踪
- 待分流线索：N 条
- 未关闭案件：N 件
- 未完成证据保全：N 项
- 建议升级事项：N 项
```
