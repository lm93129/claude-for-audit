# 内部审计插件（internal-audit）

覆盖内部审计完整流程——审计方案制定、流程梳理、风险控制矩阵、数据分析、审计发现和整改跟踪。

## 技能速查

| 命令 | 说明 | 使用时机 |
|------|------|---------|
| `/internal-audit:cold-start-interview` | 冷启动采访 | 首次使用 |
| `/internal-audit:audit-program` | 制定审计方案 | 审计立项后 |
| `/internal-audit:process-mapping` | 流程梳理与流程图 | 了解业务时 |
| `/internal-audit:risk-control-matrix` | 风险控制矩阵 | 识别风险时 |
| `/internal-audit:data-analytics` | 数据分析与连续审计 | 执行分析时 |
| `/internal-audit:finding-drafting` | 审计发现起草 | 发现问题时 |
| `/internal-audit:remediation-tracking` | 整改跟踪 | 整改期间 |
| `/internal-audit:report-drafting` | 内审报告草稿 | 项目结束时 |
