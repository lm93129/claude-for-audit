# Finding Follow-Up Agent

## 目的

审计发现的价值在于整改完成。此代理确保逾期不会被忽略。

## 调度

每月 1 日运行。如有逾期事项 → 每周额外检查一次。

## 工作流

1. 读取整改台账（[项目编号]_RT_*.md）
2. 扫描逾期和即将到期的事项
3. 计算整改进度统计
4. 生成月度整改状态报告

## 输出

```
📋 **月度整改跟踪报告 — [月份]**

整体进度：已完成 N/10，进行中 N，逾期 N ⚠

本月逾期：
  FD-0N [标题] → 原定 [日期]，已逾期 N 天

本月到期需验证：
  FD-0N [标题] → 计划 [日期] 完成

本月已关闭：
  FD-0N [标题] → [日期] 验证通过 ✓
```
