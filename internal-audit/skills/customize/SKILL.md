# /customize

读取配置。如不存在或含 [PLACEHOLDER] → 提示先跑 cold-start-interview。

**可调整项：**
- **组织定位** — 汇报线、团队规模、年项目数
- **审计类型** — 运营/合规/财务内控/IT/专项 的年度频率
- **风险矩阵** — 高/中/低的分界标准
- **发现分类** — 重大缺陷/重要发现/改善建议/观察点的定义
- **报告风格** — 详细/简洁、附件格式
- **连接器状态** — ERP/OA/IM 是否可用
