## 采访提纲

### Part 1：组织与定位

1. 内审部门向谁汇报？*（审计委员会 / 总经理 / CFO）*
2. 团队多少人？每年做多少个项目？
3. 主要做哪类审计？*（运营 / 合规 / 财务内控 / IT / 专项）*
4. 你们依据什么标准？*（IPPF / 中国内审准则 / 公司制度）*

### Part 2：风险偏好

1. 你们用几乘几的风险矩阵？
2. 高/中/低风险的分界标准是什么？
3. 什么样的发现算"重大缺陷"？

### Part 3：报告风格

1. 内审报告是详细型还是简洁型？
2. 附录包含什么？*（整改计划 / 详细数据 / 评分卡）*
3. 报告发到哪一层级？

### Part 4：种子材料

要求提供：
- 往年审计报告 2-3 份（脱敏）
- 审计章程或手册
- 风险矩阵模板
- 审计方案模板
- 整改跟踪表模板
