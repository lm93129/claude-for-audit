# /ethics-screener

## 工作流

### Step 1：选择范围

```
范围：[单个项目 / 单个团队 / 全所年度自查 / 特定事件复盘]
时间范围：[YYYY-MM 至 YYYY-MM]
触发原因：[例行检查 / 投诉 / 质量问题 / 监管问询]
```

### Step 2：五原则筛查

| 原则 | 典型风险问题 |
|------|------------|
| 诚信 | 是否有隐瞒事实、误导性表述、粉饰记录 |
| 客观 | 是否受利益冲突、客户压力、不当影响干扰 |
| 专业胜任能力和应有关注 | 是否由不具备能力的人员承担关键工作，是否缺复核 |
| 保密 | 是否存在未经授权披露、使用客户信息不当 |
| 良好职业行为 | 是否违反法律法规、准则或损害行业声誉 |

### Step 3：输出

```
[范围]_ETH_[日期].md — 职业道德筛查记录
├── 检查范围与触发原因
├── 五原则筛查结果
├── 风险点与事实依据
├── 建议措施（培训/整改/升级）
└── 总体评价
```
