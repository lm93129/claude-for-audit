# /documentation-qc

## 检查清单

| 类别 | 检查点 |
|------|------|
| 完整性 | 计划、风险评估、重要性、执行、复核、报告是否齐全 |
| 一致性 | 试算表、底稿、报告之间数字是否一致 |
| 可追溯性 | 每项结论是否能追溯到证据 |
| 及时性 | 是否在规定归档期限内完成 |
| 留痕 | 编制人、复核人、日期、版本记录是否完整 |
| 安全性 | 是否存在未脱敏信息、权限过宽、文件混乱 |

## 输出样例

```
[项目编号]_DQC_[日期].md — 底稿质控检查记录
├── 检查范围
├── 缺失文件清单
├── 勾稽/留痕问题清单
├── 归档时限风险
└── 处理建议
```
