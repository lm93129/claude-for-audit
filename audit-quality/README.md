# 审计质量监控插件（audit-quality）

覆盖审计质量管理——独立性检查、项目复核、同业互查、底稿质控、职业道德筛查。

## 技能速查

| 命令 | 说明 |
|------|------|
| `/audit-quality:cold-start-interview` | 冷启动采访 |
| `/audit-quality:independence-check` | 独立性检查 |
| `/audit-quality:engagement-review` | 项目复核 |
| `/audit-quality:peer-review` | 同业互查 |
| `/audit-quality:documentation-qc` | 底稿质量控制 |
| `/audit-quality:ethics-screener` | 职业道德筛查 |

## 插件定位

这个插件不是做某一个具体审计项目的程序执行，而是做“审计项目本身是否做得合规、做得规范、做得可复核”。

核心关注：
- 独立性是否满足
- 底稿是否完整、及时、可复核
- 重大判断是否有充分记录
- 报告是否与底稿一致
- 团队是否遵守职业道德要求
