# /analytical-review

## 重要：先加载 Practice Profile

读取 `~/.claude/plugins/config/claude-for-audit/statutory-audit/CLAUDE.md` → 审计方法论 → 分析性复核：

- 波动阈值：[默认 5%]
- 比较基准：上年同期 + 预算 + 行业数据
- 异常标记：超出阈值的项目标记为 【待解释】

## 适用范围

分析性复核可用于审计的三个阶段，通过参数选择阶段：

| 阶段 | 用途 | 参数 |
|------|------|------|
| 计划阶段 | 识别异常领域，指导实质性程序 | `--planning` |
| 实质性程序阶段 | 辅助验证细节测试结果 | `--substantive` |
| 最终复核阶段 | 评价整体合理性 | `--final`（默认）|

## 工作流

### Step 1：加载数据

- 从 ERP 或用户上传获取本年与上年财务报表
- 从 Practice Profile 读取上年审计后的预期值
- 如连接了企查查 MCP，获取行业平均数据

### Step 2：水平分析

```
科目            本年         上年     变动%    标记
──────────────────────────────────────────────────
营业收入      100,000      85,000   +17.6%    ✓
营业成本       60,000      51,000   +17.6%    ✓
销售费用        8,000      12,000   -33.3%    ⚠ 【待解释】
管理费用        5,000       6,000   -16.7%    ✓
研发费用       12,000      10,000   +20.0%    ⚠ 【待解释】
其他收益            -       5,000    -100%    ⚠ 【关注】
──────────────────────────────────────────────────
异常标记规则：
- 变动 > [波动阈值]           → ⚠ 【待解释】
- 变动 > [3倍波动阈值]        → ⚠ 【重点关注】
- 与收入变动方向相反的费用     → ⚠ 【异常】
- 无上期数据的本期新增科目     → ⚠ 【新增】
```

### Step 3：比率分析

```
毛利率：  本年 40.0% → 上年 40.0% → 持平 ✓
净利率：  本年 15.0% → 上年 16.5% → -1.5% ⚠ 【待解释】
营业费用率：本年 15.0% → 上年 21.2% → 下降 6.2% ⚠ 【待解释】

行业对比（如已连接企查查）：
- 本企业毛利率 40.0% vs 行业平均 38.0% → 正常 ✓
- 本企业费用率 15.0% vs 行业平均 20.0% → 低于行业 ⚠ 【待解释】
```

### Step 4：趋势分析（3-5年）

如提供多年数据，展示趋势：

```
收入趋势：     85,000 → 92,000 → 100,000 → CAGR +8.5%
毛利率趋势：   41.0% → 40.5% → 40.0%
净利润趋势：   14,000 → 14,800 → 15,000
```

### Step 5：生成底稿

```
[底稿编码]-AR-01 分析性复核表.xlsx
Sheet 1：水平分析表
Sheet 2：比率分析表
Sheet 3：趋势分析图（如有）
Sheet 4：异常项目跟进记录
```

每个异常项目自动生成待跟进清单：

| 科目 | 异常描述 | 计划后续审计程序 | 责任人 | 完成 |
|------|---------|----------------|-------|------|
| 销售费用 | 下降 33.3% | 检查销售费用明细账，确认是否有费用分类变更 |
| 研发费用 | 增长 20.0% | 检查研发项目立项文件，确认费用化/资本化划分 |
