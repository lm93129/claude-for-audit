# /journal-entry-testing

## 重要：先加载 Practice Profile

读取 Practice Profile → 审计方法论 → 实质性程序策略。

## 异常分录识别规则

按以下维度自动筛选异常分录：

### 时间维度
- 周末/节假日录制的凭证
- 夜间（22:00-06:00）录制的凭证
- 结账日后补录入的凭证

### 金额维度
- 取整数金额（如 1,000,000.00）
- 略低于重要性水平的金额
- 略低于审批权限的金额
- 同一金额反复出现的凭证

### 科目维度
- 借贷方为非常见对应关系：
  - 借：银行存款 → 贷：其他应付款（非主营）
  - 借：费用 → 贷：其他应付款/预提费用（无合同支持）
- 分录中非常用会计科目

### 其他红旗
- 无附件或清单的凭证
- 非标准凭证类型（红字、调整、更正）
- 编制人与审批人为同一人
- 频繁的红字冲销再补录

## 输出格式

```
异常分录编号        日期      金额(元)          异常类型              风险评估
─────────────────────────────────────────────────────────────────────
J-2024-12-31-001    12/31   5,000,000.00    周末录入+整数金额      ⚠ 关注
J-2024-12-31-002    12/31   4,800,000.00    略低于重要性水平        ⚠ 关注
J-2024-12-31-003    12/31   2,000,000.00    非常规借贷方对应关系    ⚠ 高度关注
```

每项异常分录需执行：
1. 检查原始凭证
2. 询问编制人和审批人
3. 获取合同/发票/支持性文件
4. 记录结论
