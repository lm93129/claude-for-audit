# IT审计插件（it-audit）

覆盖 IT 审计核心领域——一般控制审计、应用控制审计、数据迁移审计、网络安全评价和计算机辅助审计技术。

## 技能速查

| 命令 | 说明 |
|------|------|
| `/it-audit:cold-start-interview` | 冷启动采访 |
| `/it-audit:gc-attestation` | 一般控制审计（GC） |
| `/it-audit:application-control` | 应用控制审计 |
| `/it-audit:data-migration-audit` | 数据迁移审计 |
| `/it-audit:cybersecurity-review` | 网络安全评价（等保2.0对标） |
| `/it-audit:caatt` | 计算机辅助审计技术（CAATT） |
