# /caatt

## 典型用例

| 用例 | 输入 | 输出 |
|------|------|------|
| 权限重叠分析 | 用户-角色-权限矩阵 | 高风险权限组合清单 |
| SoD 冲突检测 | 岗位与权限清单 | 不相容职责冲突报告 |
| 变更日志分析 | 变更单/部署日志 | 未审批/紧急变更清单 |
| 日志模式识别 | 登录/操作日志 | 异常时间、异常地点、异常频率 |
| 批量对账 | 源系统/目标系统数据 | 差异记录清单 |

## 输出

```
[项目编号]_CAATT_[日期].md — CAATT 分析报告
├── 数据来源与规则说明
├── 分析脚本/逻辑摘要
├── 异常清单
├── 高风险事项
└── 建议下一步人工验证范围
```
