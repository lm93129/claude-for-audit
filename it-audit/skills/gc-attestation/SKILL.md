# /gc-attestation

## 核心模块

### 1. 访问控制
- 新增/变更/删除用户是否经过审批
- 高权限账号是否受控
- 离职禁用是否及时
- 定期权限复核是否执行

### 2. 变更管理
- 变更申请、审批、测试、上线、回退记录是否完整
- 紧急变更是否事后补批
- 生产环境变更是否受限

### 3. 运维管理
- 作业调度、批处理、异常处理是否有记录
- 关键告警是否跟进
- 接口失败是否有补偿机制

### 4. 备份与恢复
- 备份策略是否覆盖关键系统
- 备份是否验证可恢复
- 恢复演练是否定期执行

### 5. 日志与监控
- 关键日志是否开启和留存
- 安全事件是否监控、升级和闭环

## 输出

```
[项目编号]_ITGC_[日期].md — 一般控制审计报告
├── 范围和系统清单
├── 五大模块测试结果
├── 缺陷分类（🔴/🟠/🟡/🟢）
└── 整改建议与优先级
```
