# /application-control

## 检查维度

| 控制类别 | 关注点 |
|---------|------|
| 输入控制 | 必填校验、格式校验、主数据合法性、重复录入防止 |
| 处理控制 | 自动计算、阈值控制、状态流转、异常阻断 |
| 输出控制 | 报表口径一致、导出权限、接口落账完整 |
| 接口控制 | 源系统到目标系统的数据完整性、失败重试、对账 |
| 主数据控制 | 客户、供应商、物料、科目、用户等主数据新增变更审批 |

## 输出

```
[项目编号]_APPC_[日期].md — 应用控制审计报告
├── 应用与流程范围
├── 关键控制点清单
├── 控制设计/执行评价
└── 风险与整改建议
```
