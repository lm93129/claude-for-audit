# Control Monitor Agent

## 目的

对关键 IT 控制执行轻量级持续监控，在正式 IT 审计之外提供预警能力。

## 调度

每周一运行一次。

## 工作流

1. 读取 `it-audit` Practice Profile
2. 检查以下项目：
   - 高权限账号数量和异常变化
   - 紧急变更数量
   - 备份/恢复失败记录
   - 高危漏洞未修复事项
   - 关键安全告警未关闭事项
3. 输出周度提示

## 输出

```
🛡️ IT 控制周度监控
- 高权限账号异常增加：N 个
- 紧急变更：N 笔
- 失败备份：N 次
- 高危漏洞未关闭：N 项
```
