# 示例案例（Examples）

本目录提供可直接用于演示 `claude-for-audit` 的最小案例包。

当前包含：

1. `statutory-audit-manufacturing-2024/`
   - 制造业公司 2024 年度法定审计示例
   - 覆盖：risk-assessment → materiality → analytical-review → substantive-testing → report-drafting

2. `internal-audit-procure-to-pay/`
   - 采购到付款流程内部审计示例
   - 覆盖：audit-program → process-mapping → risk-control-matrix → finding-drafting → remediation-tracking

3. `compliance-audit-app-pipl/`
   - APP 用户信息处理个保法专项合规示例
   - 覆盖：regulatory-checker → pip-compliance

4. `forensic-audit-procurement-kickback/`
   - 采购回扣举报调查示例
   - 覆盖：whistleblower-triage → red-flags → transaction-analysis → data-forensics

## 使用方式

先阅读每个目录下的 `README.md`，再按其中的建议步骤调用对应插件技能。

## 注意

- 所有数据均为虚构示例，仅用于演示结构和流程。
- 不用于真实客户或真实监管场景。
- 目的是帮助快速理解插件如何落地，而不是提供行业标准答案。
