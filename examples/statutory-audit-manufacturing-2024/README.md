# 示例案例：制造业 2024 年度法定审计

## 场景简介

被审计单位：华东精工制造有限公司（虚构）
行业：制造业
年度：2024 年
会计准则：中国企业会计准则（CAS）
审计目标：跑通法定审计插件的一条完整主链路。

## 公司背景（示例）

- 主营：精密零部件生产与销售
- 客户：汽车和工业设备制造商
- 2024 年收入：1.2 亿元
- 2024 年净利润：1,200 万元
- 总资产：9,500 万元
- 存货余额较大，原材料和产成品占比较高
- 应收账款周转在年末略有变慢
- ERP 已上线，但存货盘点与成本结转仍有部分手工环节

## 示例输入数据

### 1. 简化财务摘要

| 项目 | 2024 | 2023 |
|------|------:|------:|
| 营业收入 | 120,000,000 | 108,000,000 |
| 营业成本 | 87,600,000 | 78,000,000 |
| 毛利 | 32,400,000 | 30,000,000 |
| 销售费用 | 8,400,000 | 7,900,000 |
| 管理费用 | 6,200,000 | 6,000,000 |
| 研发费用 | 4,800,000 | 4,200,000 |
| 信用减值损失 | 1,500,000 | 900,000 |
| 净利润 | 12,000,000 | 14,500,000 |
| 总资产 | 95,000,000 | 88,000,000 |
| 应收账款 | 18,000,000 | 13,500,000 |
| 存货 | 26,000,000 | 21,000,000 |
| 货币资金 | 12,500,000 | 14,800,000 |

### 2. 风险提示（可喂给 risk-assessment）

- 2024 年第四季度收入占全年 41%
- 应收账款增速（+33%）高于收入增速（+11%）
- 信用减值损失大幅增加
- 存货期末余额较高，且部分产成品周转放缓
- 本年新增 2 家前五大客户
- 年末存在几笔大额手工调账凭证

### 3. 重要性水平示例口径

建议假设：
- 基准：营业收入
- 比例：5%
- 计划重要性 PM：6,000,000
- 实际执行重要性 TE：4,500,000（75%）
- 明显微小错报 SAD：300,000（PM 的 5%）

### 4. 收入实质性程序示例线索

- 12/29~12/31 存在 4 笔大额销售
- 其中 2 笔为新客户
- 1 笔收货签收日期为 2025/01/03
- 1 笔验收单缺失

这些线索可用于测试：
- 截止性
- 新客户真实性
- 收入确认时点

### 5. 函证示例线索

- 应收账款余额前五大客户合计占比 62%
- 其中客户 A 回函一致
- 客户 B 未回函，但有期后回款
- 客户 C 回函差异 50 万，需要差异分析

## 预期演示输出

如果演示顺利，你大概率会看到以下输出方向：

1. `risk-assessment`
   - 收入确认、应收账款计价、存货计价被标为高风险或中高风险

2. `materiality`
   - 输出 PM / TE / SAD 计算表

3. `analytical-review`
   - 应收账款、信用减值、Q4 收入占比被标记为异常

4. `substantive-testing --revenue`
   - 发现跨期风险和支持性文件不足问题

5. `confirmations`
   - 形成回函差异和替代程序记录

6. `report-drafting`
   - 在无进一步重大错报的前提下，倾向无保留意见 + 关注收入确认和坏账准备作为 KAM 候选
