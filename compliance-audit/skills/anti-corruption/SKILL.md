# /anti-corruption

## 工作流

### Step 1：确定审计范围

```
范围：[销售费用 / 市场活动 / 招待礼品 / 采购招标 / 第三方渠道 / 顾问费 / 返点返利]
期间：[YYYY-MM 至 YYYY-MM]
重点对象：[高风险地区/高风险岗位/高风险第三方]
```

### Step 2：核心检查清单

| 领域 | 红旗/控制点 |
|------|------------|
| 礼品招待 | 是否超标准、是否有审批、是否有受赠对象和目的记录 |
| 第三方管理 | 是否尽调、是否签反贿赂条款、是否有异常佣金比例 |
| 采购招标 | 是否存在单一来源过多、比价记录缺失、关联供应商 |
| 费用报销 | 是否存在模糊摘要、整额、高频小额拆分、缺附件 |
| 付款控制 | 是否存在紧急付款、事后补审批、付款去向异常 |

### Step 3：高风险测试

示例输出：

```
┌─ 反商业贿赂测试结果 ─────────────────────────────────────┐
│ 检查项                     │ 异常数 │ 备注                 │
│ 超标准招待                 │ 8      │ 其中 3 笔无审批        │
│ 第三方高佣金                │ 5      │ 超过合同基准 2 倍       │
│ 单一来源采购缺比价          │ 12     │ 采购金额合计¥380万      │
│ 咨询服务费缺工作成果证明     │ 4      │ 风险较高               │
└─────────────────────────────────────────────────────────┘
```
