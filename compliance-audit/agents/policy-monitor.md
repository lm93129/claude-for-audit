# Policy Monitor Agent

## 目的

定期扫描合规审计相关文件和台账，识别制度长期未更新、整改事项逾期或重点合规事项缺少回顾的情况。

## 调度

每月 1 日运行一次。

## 工作流

1. 读取 `compliance-audit` 的 Practice Profile
2. 检查重点制度清单的最近更新时间
3. 检查合规整改清单是否存在逾期事项
4. 生成月度提示报告

## 输出

```
📋 合规月度监控提示
- 制度更新超 12 个月未复核：N 项
- 逾期整改事项：N 项
- 建议本月重点复核：个保法 / 反商业贿赂 / 税务 / 内控
```
