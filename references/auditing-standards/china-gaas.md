# 中国注册会计师审计准则（CSA）常用索引

本文件列出法定审计技能中引用的主要 CSA 准则编号，供技能编写时参考。

| 准则编号 | 名称 | 本仓库对应技能 |
|---------|------|--------------|
| CSA 1101 号 | 注册会计师的总体目标和审计工作的基本要求 | 所有技能 |
| CSA 1111 号 | 初步业务活动 | wp-generator |
| CSA 1121 号 | 对财务报表审计实施的质量控制 | audit-quality/engagement-review |
| CSA 1131 号 | 审计工作底稿 | wp-generator |
| CSA 1141 号 | 财务报表审计中与舞弊相关的责任 | journal-entry-testing, forensic-audit |
| CSA 1142 号 | 财务报表审计中对法律法规的考虑 | compliance-audit |
| CSA 1151 号 | 与治理层的沟通 | report-drafting |
| CSA 1152 号 | 向治理层和管理层通报内部控制缺陷 | ic-review |
